Persian Date Picker

Add persian datepicker to input field.

This Date picker work with Iranian calendar .

More info at: http://en.wikipedia.org/wiki/Iranian_calendar

Example

Attached to a field with the format specified via options.

Using persianDatepicker.js

Call the datepicker via javascript: 

		
  1. $('.datepicker').pDatepicker()

Options

Name type default description
viewFormat string 'mm/dd/yyyy' the date format, combination of d, dd, m, mm, yy, yyy.
viewMode string day day,month,year
persianDigit Boolean true if true all digit convert to persian digit.
position Array auto [x,y] , define a position of datepicker relative to input element.
autoClose Boolean false If true picker close When Select day
toolbox Boolean true Show Hide toolbox element
mask String | Intger false If Mask Set Datepicker add input hidden whit name of main input and set value for this on select date, acceptable value: unix,gregorian
viewFormatter Function -- format value of input
{...
viewFormatter: function(unixDate){return unixDate}
...}
maskFormatter Function -- format value of mask input
{...
maskFormatter: function(unixDate){return unixDate}
...}
