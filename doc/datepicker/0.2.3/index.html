Quick Example

Attached to a field with the format specified via options.

Qiuck Start

		<head>
			<link rel="stylesheet" href="css/pwt.datepicker.css"/>
			<script src="js/jquery.js"></script>
			<script src="js/persian.date.js"></script>
			<script src="js/persian.datepicker.js"></script>
		</head>
		
		<script type="text/javascript">
			$(document).ready(function() {
		            $(".example1").pDatepicker();
            		});
		</script>
		<input type="text" class="example1" />

Options

Name type default description
format 0.0.4 string 'mm/dd/yyyy' the date format, combination of d, dd, m, mm, yy, yyy.
formatter 0.0.4 Function -- format value of input
{...
formatter: function(unixDate){return unixDate}
...}
altField 0.0.4 selector string false An input element that is to be updated with the selected date from the datepicker. Use the altFormat option to change the format of the date within this field. Leave as blank for no alternate field. acceptable value: : '#elementId','.element-class'
altFormat 0.0.4 String unix The dateFormat to be used for the altField option. acceptable value: unix,gregorian,g,u,YYYY/mMM/DD
altFieldFormatter 0.0.4 Function -- format value of 'altField' input
{...
altFieldFormatter: function(unixDate){return unixDate}
...}
observer 0.0.4 Boolean false if you use $(element).val() to set input val and 'observer : true' , datepicker sync with your inputed Value. Observer Watch input and altField Elemets.
onShow 0.0.4 Function -- A function that takes current datepicker instance. It is called just before the datepicker is displayed.
onHide 0.0.4 Function -- A function that takes current datepicker instance. It is called just before the datepicker Hide.
onSelect0.0.4 Function -- A function that takes current datepicker unixDate. It is called When Day Select.
viewMode string day day,month,year
persianDigit Boolean true if true all digit convert to persian digit.
position Array auto [x,y] , define a position of datepicker relative to input element.
autoClose Boolean false If true picker close When Select day
toolbox Boolean true Show Hide toolbox element

maskDeprecated In 0.0.4 String | Intger false If Mask Set Datepicker add input hidden whit name of main input and set value for this on select date, acceptable value: unix,gregorian

Methods

Name Arguments Description Sample Code
show() 0.0.4 This method does not accept any arguments. Open the date picker. 
$( ".selector" ).datepicker( "show" );
hide() 0.0.4 This method does not accept any arguments. Close the date picker. 
$( ".selector" ).datepicker( "hide" );
destroy 0.0.4 This method does not accept any arguments. Removes the datepicker functionality completely. 
$( ".selector" ).datepicker( "destroy" );
